BAO CAO KIEM THU API - AssistAi.Api
Thoi gian: 2026-03-01
Moi truong test: Local (http://localhost:5188)

1) Kiem tra build
- Lenh: dotnet build .\AssistAi.Api.csproj -v minimal
- Ket qua: Thanh cong

2) Ket qua test endpoint

[1] POST /api/auth/register
- Case: Dang ky tai khoan moi
- Ket qua: 200 OK
- Response: {"message":"User registered successfully"}

[2] POST /api/auth/register
- Case: Dang ky trung username/email
- Ket qua: 400 Bad Request
- Response: {"message":"Email or Username already exists"}

[3] POST /api/auth/login
- Case: Dang nhap sai mat khau
- Ket qua: 400 Bad Request
- Response: {"message":"Username or password is incorrect"}

[4] POST /api/auth/login
- Case: Dang nhap dung thong tin
- Ket qua: 200 OK
- Response: Tra ve JWT token hop le

[5] GET /api/models
- Case: Khong gui token
- Ket qua: 401 Unauthorized

[6] GET /api/models
- Case: Gui token hop le
- Ket qua: 200 OK
- Response: Danh sach model theo tier Free (8 models)

[7] POST /api/chat/chat
- Case: Khong gui token
- Ket qua: 401 Unauthorized

[8] POST /api/chat/chat
- Case: Gui token hop le
- Ket qua: 200 OK
- Response: {"response":"Hi!"}

3) Tong ket
- Tat ca endpoint da test deu hoat dong dung theo logic hien tai.
- Xac thuc JWT cho /api/models va /api/chat/chat hoat dong dung.
